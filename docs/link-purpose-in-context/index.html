上記の例の様に、異なる段落など、概要文から少し離れた箇所に「続きを読む」だけをリンクテキストとしたリンクがある場合、前後の文脈を考慮してもリンクの目的が不明瞭と判断されると考えます。

概要文のすぐ後ろ（同じ段落）にリンクがあれば、「今まで読んできた概要文」の「続きを読む」なんだなと理解できると思いますが、離れてしまうとそういったことがわかりにくくなるので注意が必要です。

「ニュースタイトル」の記事全文を読む

改善案（1）は概要文のすぐ後ろ（同じ段落）にリンクを置いた例です。前述の通り、この配置であれば「今まで読んできた概要文」の「続きを読む」なんだなと理解できると思います。

改善案（2）は、リンク単体でもリンクの目的がわかるようにした例です。本来はこのように、リンク単体だけでなんのためのリンクなのかがわかるようにするのが望ましいでしょう。もちろん aria-label 属性などを併用しても良いです。

上記のような例の場合であれば、見出しと前段のテキストから、リンクの目的は理解できると考えても問題ないでしょう。各リンクに書籍名を繰り返し入れると冗長な場合などは直近の見出しなどでリンクの目的がわかるようにしましょう。

惜しい例

詳しいお申し込み方法は、「●●大会への申込み手順」のページに記載しておりますのでそちらをご覧ください。

●●大会への申込み手順はこちら。

改善案

詳しいお申し込み方法は、「●●大会への申込み手順」のページに記載しておりますのでそちらをご覧ください。

●●大会への申込み手順はこちら。

最初の例でも前後の文脈を考えれば（好ましくはないものの）達成基準を満たすと考えることもできますが、どこにリンクを張るかを工夫することで「単体でも」わかりやすいリンクになります。常に想像力を働かせましょう。
